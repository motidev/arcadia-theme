### ⚙ Recommended Settings 
  ```jsonc

    "VsCode": {
      
        "workbench.colorTheme": "Dyno Dark Theme",
        "editor.fontSize": "14px",
        "editor.fontFamily": "Cascadia Code PL"
    }

  ```

Download the FontFamily [**Cascadia Code**](https://github.com/microsoft/cascadia-code/releases).
