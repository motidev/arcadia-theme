### ⚙ Recommended Settings 
  ```jsonc

    "VsCode": {
        "workbench.colorTheme": "Arcadia Theme",
        "editor.fontSize": "14px",
        "editor.fontFamily": "Cascadia Code PL"
    }

  ```

Download the [**Cascadia Code Font**](https://github.com/microsoft/cascadia-code/releases).
